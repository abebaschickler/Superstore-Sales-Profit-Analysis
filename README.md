# Superstore-Sales-Profit-Analysis
Retail sales and profitability analysis using SQL, Power Query, and Tableau.
## Project Overview
This project analyzes the Sample Superstore retail dataset to investigate why the Central region generates lower profitability than the other regions. Using Power Query for data cleaning, SQL Server for analysis, and Tableau for visualization, the project identifies the key drivers of poor financial performance and provides business recommendations.
## Business Problem
The company experienced lower profitability in the Central region compared with other regions. This analysis investigates the factors contributing to this performance gap and identifies the categories and subcategories responsible for profit losses.
## Objective of the Analysis
The objective of this project is to analyze regional, category, and subcategory
performance to identify the main drivers of low profitability and provide data-driven
recommendations for improving business performance.
## Dataset
The dataset used in this project is the Sample Superstore dataset, a publicly available retail sales dataset commonly used for data analytics practice and visualization projects.
### Data Preparation
- Removed duplicate records
- Checked missing values
- Verified data types
- Trimmed text fields

## Key Findings 
- The West region achieved the highest sales and profit.
- The Central region had the lowest profit margin despite relatively high sales.
- High discount levels in the Central region were associated with lower profitability.
- Furniture subcategories such as Tables, Bookcases, and Furnishings contributed most to the losses
